FULL STACK DEVELOPMENT/TECH LEAD


Location: Cochin


Who are we looking for




We are looking for FullStack Development/Tech Lead, responsible for the coding, innovative design and layout of our ground up product implementation in Healthcare domain. FullStack developer responsibilities include building our product from UI to the java / scala based core components. From Requirement Analysis to Design to hands on coding the core interfaces to APIS to Core components - all the way to completion from the bottom up, fashioning everything from the home page to site layout and function.

Preference: Currently Working in a startup or Should have worked with a startup for at least 2 years.



What will you be doing?


  • Write well designed, testable, efficient code by using best software development practices.

  • Develop backend Services and APIs.

  • Candidate must have implemented the SOAP web services and REST services.

  • Gather and refine specifications and requirements based on technical needs.

  • Create and maintain software documentation.

  • Stay plugged into emerging technologies/industry trends and apply them into operations and activities.

  • Cooperate with web developers in integrating the APIs and services.

  • Lead a team of 3 to 4 developers.

  • Represent the Development team in Sprint Standup meeting and highlight impediments with the stakeholders.



Requirement



  • 3 to 4 years Experience.

  • Has strong experience in Core Java/ Python / Scala.

  • Completely hands-on with development and code writing.

  • Netbeans / Eclipse IDE.

  • Good understanding of HTML5, JQuery, ReactJS+ Redux.

  • Good understanding of Webservers and Application Servers: Tomcat, Netty, etc..

  • Clear understanding, coding and design experience in either combination: REST services & JSON usage OR SOAP & XML (OR BOTH), Sync and Async Services.

  • Candidate must have implemented the web services, but also have clear understanding of the API design.

  • Has worked on frameworks such as Spring, Play Framework, and can articulate the parts and how it can be leveraged.

  • Knowledge on MongoDB and MySQL.

  • Understands DevOps processes. Knows CI and CD practices. Has worked in projects where CI was setup..

  • Understands and able to articulate concepts such as Service Orientation, Service Aggregation, Service Orchestration.

  • Understand what is clean code and be able to help others in this spaceUnderstanding of messaging and/or asynchronous processing. Has applied asynchronous messaging patterns in the past.

  • Articulates trends and concepts such as MicroServices, Service Discovery, Containerization.

  • Understands the operational aspects of a platform.

  • Aware of Continuous delivery.

  • Proven history of taking up adoption of open source frameworks, learning the usage and implementing it as needed.

  • Knowledge on Unit Test Scripts and Suites.

  • Good knowledge in multi- threading and facade creation for external third party libraries.

  • Knowledge on AWS cloud is an added advantage.

  • Knowledge on Big data hadoop is an added advantage.

  • Knowledge on Web Page / Payload performance and optimization is an added advantage

  • Ability to work and thrive in a fast-paced agile environment, learn rapidly and master diverse web technologies and techniques.

  • Your motivation expressed in a form of your choice.
